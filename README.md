# BFS/DFS Adventure (Binary Trees)

This repo is a progressive mini adventure for first year students.
Solve one level to unlock the next.

## Install
Python 3 only. No dependencies.

## How to play
From the repo folder:

- See your level:
  python3 runner.py status

- Run current level tests:
  python3 runner.py test

- Reveal next mission after passing:
  python3 runner.py reveal

## Rules
- Edit only: `src/levels/levelX.py`
- Do not edit tests.
- Use basic Python.
- Keep functions simple and readable.

## Structure
- `src/structures/` : BinTree and Queue
- `src/levels/`     : student tasks
- `hints/`          : hints per level
- `missions/`       : story text revealed per level
- `tests/`          : test suite
- `.progress/`      : local progress markers

Good luck.
